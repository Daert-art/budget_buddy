Budget Buddy

Твій фінансовий помічник для розумного управління бюджетом

Мета проєкту

Budget Buddy — це персональний фінансовий помічник, створений для того, щоб допомогти вам ефективно керувати своїми доходами та витратами.

Цільова аудиторія

- Приватні користувачі, які хочуть краще контролювати особисті фінанси
- Родини, які планують спільний бюджет
- Студенти, фрілансери, працівники будь-яких сфер

Основні функції

  • Реєстрація та авторизація користувачів
  • Створення рахунків (гаманців)
  • Додавання операцій (доходів / витрат)
  • Категорії та теги для операцій
  • Планування бюджетів по категоріях
  • Підтримка повторюваних операцій
  • Графіки та прогрес-бари по бюджетах
  • Гарний, зручний та сучасний інтерфейс

Використані технології

  • Python 3.13 + Django 5.2
  • SQLite / PostgreSQL
  • HTML5, CSS3 (кастомні стилі)
  • Шаблонізація Django Template Language (DTL)
